Library Website – Project Week 2
Een moderne, gebruiksvriendelijke bibliotheekwebsite ontwikkeld tijdens Project Week 2. Het doel van dit project is het ontwerpen en bouwen van een duidelijke, toegankelijke interface waar bezoekers boeken kunnen ontdekken, de catalogus kunnen doorzoeken en meer kunnen lezen over de diensten van de bibliotheek.

Projectdoelen
Een overzichtelijke en intuïtieve website-indeling maken

Bezoekers eenvoudig boeken laten bekijken en zoeken

Bibliotheekinformatie gestructureerd en visueel aantrekkelijk presenteren

Vaardigheden in HTML, CSS en JavaScript toepassen

Werken volgens de projectvereisten

Functionaliteiten
Homepage met introductie en snelle navigatie

Boekenoverzichtspagina met categorieën of volledige catalogus

Detailpagina per boek met beschrijving, auteur en beschikbaarheid

Zoekfunctie (indien geïmplementeerd)

Responsief ontwerp voor desktop en mobiel

Strakke, moderne UI geïnspireerd op echte bibliotheekwebsites


Gebruikte technologieën
HTML5

CSS3

JavaScript
